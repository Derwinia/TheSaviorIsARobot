C'est sur cette page que vous pourrez trouver une série d'information concernant la pollution de l'air a travers le monde

Ce site vous donne également la possibilité de comparer les différentes informations entre pays ou encore différentes date

Vous pouvez cliquer sur la map ci-dessous pour récupérer une coordonées précises ou vous pouvez sélectionner parmis une liste que j'ai déja pré-selectionné pour vous !

Il est nécessaire de cliquer sur le bouton de confirmation situé juste a droite pour récuperer les informations de la composition de l'air a l'heure de demander, si non préciser, l'heure par défaut sera de midi!
